# MIDNIGHT CHAT PARTY — ステッカー販売管理

ライブブースでのステッカー販売を、スマホからその場で管理するためのWebアプリ。

**公開URL**: https://isojz.github.io/midnight-chat-sales/

## できること
- **会計**: デザインA/B × 通常(¥300)/SNS割引(¥200) をタップで記録。複数まとめ会計OK。
- **集計**: 総売上・販売枚数・A/B別・通常/割引別・割引率・残り在庫をリアルタイム表示。
- **履歴**: 各会計の取り消し（在庫も自動で戻る）。
- **在庫**: 初期在庫から販売枚数を引いて残数を自動計算。残りわずか/在庫切れを色で警告。
- **CSV**: 全記録をエクスポート（バックアップ・集計用）。

## 使い方
1. スマホのブラウザで上記URLを開く。
2. **ホーム画面に追加** すると、アプリのように全画面・オフラインで動く。
3. 「設定」で初期在庫と単価を入れて保存。
4. 本番は「会計」タブでタップ → 「会計を記録」。

## 仕様メモ
- データは**端末内（localStorage）に保存**。サーバには送らない。ブラウザのデータを消すと消えるので、節目で **CSVエクスポート** を取ること。
- オフライン対応（Service Worker）。一度開けば電波が無くても起動する。
- 単一端末・単一ユーザー運用前提（同時編集の同期は無し）。
