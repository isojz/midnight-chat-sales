# MIDNIGHT CHAT PARTY — 物販管理

ライブブースでの物販（ステッカー・グッズ）を、スマホからその場で管理するためのWebアプリ。

**公開URL**: https://isojz.github.io/midnight-chat-sales/

## できること
- **商品を自由に登録**: 商品ごとに「名前・通常価格・割引価格(任意)・在庫管理ON/OFF」を設定。ステッカー追加も缶バッジ/Tシャツ/CDも同じ仕組み。割引価格を入れた商品は「通常／割引」2ボタン、空欄なら1ボタン。在庫管理OFFは枚数を数えない（CD委託等）。
- **ライブ単位の管理**: 「日付＋ライブ名」でライブを登録し、会計を各ライブに紐付け。会計画面上部のバーでライブを切替。
- **会計**: 商品をタップで記録。複数まとめ会計OK。
- **集計**: 「このライブ／全期間」を切替。総売上・販売点数・商品別・通常/割引別・割引率・残り在庫＋ライブ別内訳。
- **在庫**: 商品×ライブごとに初期在庫を持ち、販売点数を引いて残数を自動計算。新規ライブの初期在庫は前ライブの残数を引き継ぐ（編集可）。残りわずか/在庫切れは色で警告。
- **履歴**: ライブで絞り込み。各会計の取り消し（在庫も自動で戻る）。
- **CSV**: 全記録をエクスポート（日付・ライブ名・商品の列＋ライブ別／商品別集計付き）。

## 使い方
1. スマホのブラウザで上記URLを開く。
2. **ホーム画面に追加** すると、アプリのように全画面・オフラインで動く。
3. 「設定 → 商品」で売る商品を登録（初期はステッカーA/B入り）。
4. 会計画面の📅バー（または「設定 → ライブ管理」）から、その日のライブを登録（日付・名前・商品ごとの初期在庫）。
5. 本番は「会計」タブで商品をタップ → 「会計を記録」。別のライブの日は📅バーで切替 or 新規追加。

## 仕様メモ
- データは**端末内（localStorage）に保存**。サーバには送らない。ブラウザのデータを消すと消えるので、節目で **CSVエクスポート** を取ること。
- オフライン対応（Service Worker）。一度開けば電波が無くても起動する。
- 単一端末・単一ユーザー運用前提（同時編集の同期は無し）。
