# MIDNIGHT CHAT PARTY — 収支管理

ライブブースでの物販に加え、チケット収入・会場費・打ち上げ等の**バンド帳簿**をスマホからその場で管理するためのWebアプリ。

**公開URL**: https://isojz.github.io/midnight-chat-sales/

## できること

### 物販
- **商品を自由に登録**: 「名前・通常価格・割引価格(任意)・在庫管理ON/OFF」を設定。割引価格を入れた商品は「通常／割引」2ボタン、空欄なら1ボタン。在庫管理OFFは枚数を数えない（CD委託等）。
- **ライブ単位の管理**: 「日付＋ライブ名」で登録、会計はライブに紐付け。ライブバーで切替。
- **在庫**: 商品×ライブごとに初期在庫、販売点数を引いて自動計算。新規ライブは前ライブの残数を引き継ぐ。
- **会計**: 商品タップで記録、まとめ会計・取消可。

### 帳簿
- **収入/支出を1タップで記録**: 収入（チケット・その他）と支出（会場費・機材/レンタル・交通費・打ち上げ・制作費・その他）を、大きなボタンからカテゴリ→金額→メモの流れで入力。
- **点検チェックリスト（新）**: ライブごとに標準7項目（物販・チケット・会場費・機材/レンタル・交通費・打ち上げ・制作費）の記録有無を✅/❌で表示。未記録項目は「＋記録」ワンタップで該当カテゴリの記帳モーダルが開く。ライブ後の点検に便利。
- **カテゴリを自由に追加**: 標準カテゴリの他に「スタジオ代」「配信費」など好きなカテゴリを追加可能。
- **ライブ紐付け or 共通経費**: 記帳ごとに「どのライブに紐付けるか／共通経費として計上するか」を選べる。
- **グッズ物販は自動集計**: 「グッズ物販」カテゴリは会計の売上を自動で反映（手動入力不要）。

### メンバー精算（新）
- **立て替えた人／受け取った人を記録**: 記帳ごとに「立替者」または「受取者」を選択。デフォルトは「バンド共通」。
- **精算サマリ**: 集計タブで、メンバーごとに「立替−受取＝バンドから返してもらう額」を色分け表示。ライブ単位／全期間で切替可能。
- **設定タブでメンバー追加**: 何人でも登録可能。

### 集計
- 「このライブ／全期間」切替。
- **収支サマリ**（差引・収入合計・支出合計）とカテゴリ別内訳。
- 物販詳細（総売上・販売点数・商品別・通常/割引別）＋ライブ別収支一覧。
- **CSV**: 物販明細・帳簿明細・ライブ別収支・全期間合計を一括エクスポート。

## 使い方
1. スマホのブラウザで上記URLを開く。
2. **ホーム画面に追加** すると、アプリのように全画面・オフラインで動く。
3. 「設定 → ライブ管理」でその日のライブを登録。
4. 本番:
   - 物販は「会計」タブで商品をタップ→「会計を記録」。
   - チケ収入・会場費・打ち上げ等は「帳簿」タブで＋収入／＋支出をタップして記録。
5. 「集計」タブで収支を確認、CSVで保存。

## 仕様メモ
- データは**端末内（localStorage）に保存**。サーバには送らない。節目で **CSVエクスポート** を取ること。
- オフライン対応（Service Worker）。一度開けば電波が無くても起動する。
- 単一端末・単一ユーザー運用前提。
- 旧バージョン（v1/v2/v3/v4）のデータは自動で最新形式にマイグレーションされる。
