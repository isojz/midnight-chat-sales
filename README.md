# MIDNIGHT CHAT PARTY — ステッカー販売管理

ライブブースでのステッカー販売を、スマホからその場で管理するためのWebアプリ。

**公開URL**: https://isojz.github.io/midnight-chat-sales/

## できること
- **ライブ単位の管理**: 「日付＋ライブ名」でライブを登録し、会計を各ライブに紐付け。会計画面上部のバーでライブを切替。
- **会計**: デザインA/B × 通常(¥300)/SNS割引(¥200) をタップで記録。複数まとめ会計OK。
- **集計**: 「このライブ／全期間」を切替。総売上・販売枚数・A/B別・通常/割引別・割引率・残り在庫＋ライブ別内訳。
- **在庫**: ライブごとに初期在庫を持ち、販売枚数を引いて残数を自動計算。新規ライブの初期在庫は前ライブの残数を引き継ぐ（編集可）。残りわずか/在庫切れは色で警告。
- **履歴**: ライブで絞り込み。各会計の取り消し（在庫も自動で戻る）。
- **CSV**: 全記録をエクスポート（日付・ライブ名の列＋ライブ別集計付き）。

## 使い方
1. スマホのブラウザで上記URLを開く。
2. **ホーム画面に追加** すると、アプリのように全画面・オフラインで動く。
3. 会計画面の📅バー（または「設定 → ライブ管理」）から、その日のライブを登録（日付・名前・初期在庫）。
4. 本番は「会計」タブでタップ → 「会計を記録」。別のライブの日は📅バーで切替 or 新規追加。

## 仕様メモ
- データは**端末内（localStorage）に保存**。サーバには送らない。ブラウザのデータを消すと消えるので、節目で **CSVエクスポート** を取ること。
- オフライン対応（Service Worker）。一度開けば電波が無くても起動する。
- 単一端末・単一ユーザー運用前提（同時編集の同期は無し）。
